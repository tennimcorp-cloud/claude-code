# Kanban To Do List App

A simple static Kanban board for managing tasks in three columns: `To Do`, `In Progress`, and `Completed`.

## Features

- Add tasks with title, description, priority, tags, and status.
- Drag and drop task cards between columns.
- Edit existing tasks.
- Responsive layout for desktop and smaller screens.

## Run locally

Open `index.html` in your browser.

## Files

- `index.html` — main page markup
- `styles.css` — UI styling
- `script.js` — app logic and drag/drop behavior
